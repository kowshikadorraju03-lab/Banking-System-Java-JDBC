## Project Overview

The Banking Management System is a desktop application developed using **Java Swing**, **JDBC**, and **MySQL**. It provides a secure and user-friendly interface to perform basic banking operations such as login authentication, deposits, withdrawals, balance enquiry, money transfer, mini statements, and CSV export.

## How to Run

1. Import the project into IntelliJ IDEA.
2. Start MySQL Server.
3. Open MySQL Workbench.
4. Execute the `database.sql` file.
5. Update the MySQL username and password in `DBConnection.java`.
6. Run `Main.java`.
7. Login using a valid username and PIN.
